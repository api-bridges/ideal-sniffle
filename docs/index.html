Authentication

Bearer Token Authentication

All requests must include your API key in the Authorization header as a Bearer token. Your key is available from the API Keys dashboard after registration.

Header Authorization: Bearer gsga_xxxxxxxxxxxxxxxxxxxxxxxxxxxx

Keep your API key secret. Never expose it in client-side JavaScript, commit it to a repository, or share it publicly. Use environment variables in all production deployments.

Rate Limits & Quotas

No Rate Limits

Get Set Go API does not impose artificial rate limits on any plan. We route requests to the fastest available provider instance and scale capacity automatically. There are no RPM (requests per minute) caps, no TPM (tokens per minute) ceilings, and no daily quotas.

Individual model providers may have their own capacity constraints during peak periods. In those cases, our router automatically falls back to equivalent model instances or queues requests with minimal latency. We surface provider-side throttling errors with a 503 status code and include a Retry-After header.

Error Codes

HTTP Status Codes

The API uses standard HTTP status codes. Error responses include a JSON body with a error.message field describing the issue in plain language.

200
OK — Request Successful
The request was processed successfully. The response body contains the model output in the standard format.
400
Bad Request — Invalid Parameters
The request body is malformed or contains invalid parameters. Check the error message for details on which field caused the issue.
401
Unauthorized — Invalid API Key
The API key is missing, expired, or invalid. Ensure you're passing the key as a Bearer token in the Authorization header.
404
Not Found — Unknown Model or Endpoint
The requested model ID or endpoint path does not exist. Use GET /v1/models to see the full list of valid model identifiers.
429
Too Many Requests — Upstream Throttle
An upstream provider is temporarily capacity-constrained. The response includes a Retry-After header. Implement exponential backoff in your client.
500
Internal Server Error
An unexpected error occurred in our infrastructure. These are rare — check our status page and retry. If persistent, contact support@getsetgoapi.com.
503
Service Unavailable — Model Offline
The requested model is temporarily unavailable from all provider sources. Try an equivalent model or wait for the Retry-After period to elapse.
